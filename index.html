About This Project

This project demonstrates a robust data processing pipeline using Python, Pandas, and GitHub Actions. It's designed to take raw Excel data, process it, and generate a structured JSON output, all automated via CI/CD.

Key components include a Python script for data manipulation, an Excel dataset as input, and a CSV intermediate format.

Core Components

execute.py

The central Python script responsible for all data processing logic. It reads from data.xlsx, converts it to data.csv, performs necessary transformations, and outputs results into result.json. This script has been carefully reviewed and corrected to ensure data integrity and accurate computations.

data.xlsx

The primary input dataset in Excel format. This file serves as the raw data source for the processing pipeline. It is automatically converted to data.csv as an intermediate step within the workflow.

Automated CI/CD Workflow

The project leverages GitHub Actions to automate linting, testing, and deployment. On every push, the workflow ensures code quality with Ruff, executes the data processing script, and publishes the resulting result.json to GitHub Pages for easy access.

This setup guarantees that the published output is always up-to-date and reflects the latest data and logic.
